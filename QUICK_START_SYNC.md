# 🎯 БЫСТРАЯ ИНСТРУКЦИЯ: Синхронизация селфи (ПК + Телефон)

## ⚠️ ГЛАВНАЯ ПРИЧИНА: Разные адреса сервера

```
❌ НЕПРАВИЛЬНО:
ПК открывает:      http://localhost:8000/gallery.html
Телефон открывает: http://localhost:8000/gallery.html ← это НЕ видит селфи с ПК!

✅ ПРАВИЛЬНО:
ПК открывает:      http://localhost:8000/gallery.html (или IP адрес)
Телефон открывает: http://192.168.1.100:8000/gallery.html ← ВИДИТ селфи!
```

---

## 🚀 За 3 минуты до рабочей синхронизации

### Шаг 1: Выясните IP адрес ПК (30 сек)

**На ПК в PowerShell напишите:**
```powershell
ipconfig
```

**Найдите в выводе:**
```
IPv4 Address . . . . . . . . . . . : 192.168.1.100
                                     ↑
                              Запомните это число!
```

### Шаг 2: Убедитесь что сервер работает (10 сек)

**В одном из терминалов должно быть:**
```
🚀 CodePath Academy запущен на http://localhost:8000
```

Если нет - запустите:
```bash
cd "c:\Users\ххх\Desktop\SCHOOL9"
node server.js 8000
```

### Шаг 3: На ПК тестируйте с localhost (1 мин)

**Браузер на ПК:**
```
→ Откройте http://localhost:8000/selfi.html
→ Нажмите "✓ Продолжить"
→ Делается селфи
→ Переходит на главную базу

Потом в другом браузере:
→ Откройте http://localhost:8000/gallery.html
→ Введите пароль: пароль123
→ Видите свое селфи ✅
```

### Шаг 4: На ТЕЛЕФОНЕ используйте IP адрес (1 мин)

**Браузер на телефоне:**
```
→ Откройте http://192.168.1.100:8000/gallery.html
   (замените 192.168.1.100 на ВАШ IP из шага 1!)
→ Введите пароль: пароль123
→ Вижу селфи с ПК ✅
```

---

## ✅ Если все работает

**Вы должны видеть:**
- На ПК браузер №1: селфи с ПК
- На ПК браузер №2: селфи с ПК
- На телефоне: селфи с ПК
- На другом телефоне: селфи с ПК и с первого телефона

---

## 🔧 Если НЕ работает

### Телефон не видит селфи?

1. **Проверьте IP адрес**
   ```powershell
   ipconfig  ← выполните еще раз
   ```
   Убедитесь что ввели правильно:
   ```
   http://192.168.1.100:8000/gallery.html ← должна быть правильная цифра
   ```

2. **Проверьте WiFi**
   - На ПК: какая сеть? (посмотрите в системном трее)
   - На телефоне: подключен к такой же сети?
   - Часто: на ПК WiFi, на телефоне мобильный → не работает!

3. **Перезагрузите сервер**
   ```
   Ctrl+C в консоли Node.js
   (ждите пока закроется)
   node server.js 8000
   ```

4. **Проверьте что сервер слушает**
   ```powershell
   netstat -an | findstr "8000"  ← должна быть строка с 8000
   ```

## 📊 Проверка что на сервере

**Откройте браузер и перейдите:**
```
http://192.168.1.100:8000/api/selfies
```

**Вы увидите JSON:**
```json
[
  {
    "id": "selfie-xxx",
    "device": "Desktop",
    "timestamp": "2024-03-14...",
    "photo": "data:image/jpeg..."
  }
]
```

- Если пусто `[]` → селфи не сохраняются
- Если есть данные → все работает!
