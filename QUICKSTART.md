# 🚀 Быстрый старт - CodePath Academy

Запуск платформы для обучения C++ в несколько шагов.

## Способ 1: Без сервера (Самый быстрый)

### Windows
1. Откройте папку `SCHOOL9`
2. Найдите файл `index.html`
3. Кликните дважды на `index.html` → откроется в браузере
4. Готово! Вы в приложении

### macOS / Linux
```bash
# Перейдите в папку проекта
cd ~/Desktop/SCHOOL9

# Откройте в браузере
open index.html  # macOS
xdg-open index.html  # Linux
```

### VS Code + Live Server
1. Откройте папку `SCHOOL9` в VS Code
2. Установите расширение "Live Server" (Go Live)
3. Кликните правой кнопкой на `index.html`
4. Выберите "Open with Live Server"
5. Браузер откроется автоматически на `http://localhost:5500`

---

## Способ 2: С Node.js сервером

### Предварительные требования
- Node.js 12+ установлен

### Установка и запуск

```bash
# 1. Перейти в папку проекта
cd ~/Desktop/SCHOOL9

# 2. Запустить сервер
node server.js

# Или через npm (если установлены зависимости)
npm start
```

**Вывод:**
```
🚀 CodePath Academy запущен на http://localhost:8000
📚 Откройте браузер и перейдите на http://localhost:8000
⏹  Для остановки сервера нажмите Ctrl+C
```

### Использование другого порта

```bash
node server.js 3000    # Порт 3000
node server.js 8080    # Порт 8080
```

## 🎯 Первые шаги в приложении

### После открытия в браузере:

1. **На главной странице (Home)**
   - Вы видите красивый hero раздел
   - Нажмите "Начать обучение" или откройте меню

2. **Перейдите на страницу курсов (Courses)**
   - Видите 3 курса: Beginner, Intermediate, Advanced
   - Выберите "C++ для начинающих"
   - Нажмите кнопку "Начать"

3. **На странице урока (Lesson)**
   - Вы видите контент урока
   - На боку: прогресс и "Что вы выучили"
   - Внизу: кнопка "Отметить как пройденный"

4. **Отслеживайте прогресс (Progress)**
   - Откройте "Прогресс" в меню
   - Видите круговую диаграмму с общим прогрессом
   - По уровням видите детальный прогресс

## 💾 Ваш прогресс

### Где хранится?
- Локально в браузере (localStorage)
- Автоматически сохраняется при отметке урока

### Как экспортировать?
Откройте DevTools (F12) и выполните:
```javascript
exportProgress();
```

Файл загрузится как JSON.

### Как очистить?
```javascript
clearAllProgress();
```

## 🔧 Расширение платформы

### Добавить новый урок

1. Откройте `src/data/courses.json`
2. Найдите нужный курс
3. В массив `lessons` добавьте новый урок:

```json
{
    "id": "lesson-1-8",
    "title": "Название вашего урока",
    "duration": 20,
    "sections": [
        {
            "title": "Раздел 1",
            "content": "Ваше объяснение...",
            "code": "// Ваш код C++"
        }
    ],
    "learned": [
        "Концепция 1",
        "Концепция 2"
    ]
}
```

4. Сохраните файл
5. Обновите страницу в браузере

### Добавить новый курс

1. Откройте `src/data/courses.json`
2. В массив корня добавьте новый курс:

```json
{
    "id": "cpp-expert",
    "title": "C++ Эксперт",
    "description": "Для опытных разработчиков",
    "level": "advanced",
    "icon": "🔥",
    "lessons": [
        {
            "id": "lesson-4-1",
            "title": "Первый урок",
            ...
        }
    ]
}
```

## 🆘 Проблемы и решения

### "CORS ошибка при загрузке JSON"
**Решение:** Используйте сервер (способ 2, 3 или 4), не открывайте `file://`

### "Браузер говорит 'Страница не найдена'"
**Решение:** 
- Проверьте правильность пути
- Убедитесь, что сервер запущен
- Откройте `http://localhost:8000` (не `file://`)

### "Прогресс не сохраняется"
**Решение:**
- Проверьте, разрешены ли cookies и localStorage
- Попробуйте приватный режим (может помочь)
- Проверьте консоль (F12) на ошибки

### "CSS стили не загружаются"
**Решение:**
- Обновите страницу (Ctrl+Shift+Del)
- Очистите кеш браузера
- Используйте сервер вместо file://

### "Код не копируется"
**Решение:**
- Браузер должен разрешить доступ к буферу обмена
- Сайт должен быть открыт через HTTP/HTTPS или localhost
- Попробуйте другой браузер
