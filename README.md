# CodePath Academy - Платформа для обучения C++

Production-ready образовательная платформа для изучения C++ от нуля до продвинутого уровня.

## 🎯 Особенности

✨ **Современный дизайн**
- Минималистичный интерфейс в стиле Stripe/Vercel
- Темная и светлая тема с плавным переходом
- Полностью адаптивный (mobile-first)

🎨 **Богатые анимации**
- CSS-анимации для плавных переходов
- GSAP интеграция для продвинутых эффектов
- Параллакс-эффекты при скролле
- Hover-анимации на всех интерактивных элементах

📚 **Реальный контент**
- 18+ полных уроков по C++
- 3 уровня сложности (Beginner, Intermediate, Advanced)
- Практические примеры кода
- Подробные объяснения на русском языке

🎓 **Функциональность**
- Отслеживание прогресса пользователя (localStorage)
- Навигация по курсам и урокам
- Красивые code-блоки с подсветкой синтаксиса
- Сохранение прогресса между сеансами

⚡ **Технологический стек**
- HTML5 + CSS3
- Vanilla JavaScript (без зависимостей)
- GSAP для анимаций
- Mobile-responsive design

## 💾 Управление прогрессом

Прогресс автоматически сохраняется в localStorage:
- Какие уроки завершены
- Время прохождения каждого урока
- Настройки пользователя (тема)

Для экспорта прогресса используйте функцию `exportProgress()` в консоли.

## 🔧 Кастомизация

### Добавить новый урок

Отредактируйте `src/data/courses.json`:

```json
{
    "id": "lesson-id",
    "title": "Название урока",
    "duration": 20,
    "sections": [
        {
            "title": "Раздел 1",
            "content": "Описание...",
            "code": "// код C++"
        }
    ],
    "learned": [
        "Что вы выучили 1",
        "Что вы выучили 2"
    ]
}
```

### Добавить новый курс

Добавьте объект в массив в `src/data/courses.json` с полной структурой курса.

### Изменить цветовую схему

Отредактируйте CSS переменные в `src/css/variables.css`:

```css
:root {
    --accent-primary: #вашцвет;
    --accent-secondary: #вашцвет;
    /* и другие */
}
```

## ⚙️ Технические детали

### JavaScript функции

**Навигация:**
- `handleNavClick(page)` - переключение страниц
- `openCourse(courseId)` - открыть курс
- `displayLesson(courseId, lessonId)` - показать урок

**Прогресс:**
- `markLessonComplete()` - отметить урок как завершенный
- `getProgress()` - получить весь прогресс
- `getCourseProgress(courseId, total)` - прогресс курса

**Анимации:**
- `setupScrollAnimations()` - инициализировать GSAP
- `triggerScrollAnimations()` - запустить анимации

### CSS Классы

- `.page` - контейнер страницы
- `.card` - элемент карточки
- `.btn` - кнопка
- `.hero` - герой секция
- `.lesson-container` - контейнер урока

## 🐛 Известные ограничения

- Без backend-сервера (используется localStorage)
- Прогресс сохраняется локально на устройстве
- Нет пользовательских аккаунтов
