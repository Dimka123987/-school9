# 📷 Инструкция по тестированию СЕЛФИ

## ✨ Что изменилось

**Вместо скриншота экрана теперь используется РЕАЛЬНОЕ СЕЛФИ с камеры!**

Приоритет захвата:
1. **📷 Первый вариант** → Реальное селфи с фронтальной камеры (всегда!)
2. **🖼️ Второй вариант** → Скриншот экрана (если камера недоступна)
3. **🎨 Третий вариант** → Цветная заглушка (если ничего не работает)

---

## 🧪 Как тестировать на разных устройствах

### 1️⃣ На смартфоне (iPhone/Android)

```
1. Откройте браузер
2. Перейдите на http://YOUR_IP:8000/selfi.html
3. Разрешите доступ к камере (появится запрос)
4. Нажмите "✓ Продолжить"
5. Секунд 2-3:
   - Будет включена фронтальная камера
   - Будет захвачено селфи
   - Селфи отправится на сервер
6. Вы перейдете на главную страницу index.html
```

**✅ Селфи успешно:** 
- В консоли (F12) увидите: `📷 Селфи захвачено с камеры`
- Фото будет сохранено в галерее

**⚠️ Если камера не сработала:**
- Консоль покажет: `⚠️ Камера недоступна: ...`
- Вместо селфи будет использован скриншот
- Вместо скриншота будет цветная заглушка

### 2️⃣ На компьютере (Desktop/Laptop)

```
1. Откройте браузер (нужна веб-камера!)
2. Перейдите на http://localhost:8000/selfi.html
3. Разрешите доступ к камере (появится диалог)
4. Нажмите "✓ Продолжить"
5. Вы увидите как:
   - Активируется веб-камера
   - Захватывается ваше селфи
   - Фото отправляется на сервер
```

**Если веб-камеры нет:** автоматически будет сделан скриншот

### 3️⃣ На старых браузерах или без камеры

```
1. Браузер не поддерживает getUserMedia?
   → Автоматически будет скриншот

2. Скриншот не работает (html2canvas не загружена)?
   → Будет цветная заглушка с иконкой устройства
```

            │
            └─ ОШИБКА?

                  └→ Canvas с иконкой и временем
                     └→ JPEG заглушка
           ↓
┌─────────────────────────────────────────┐
│ 4. Отправить на сервер (/api/selfies)  │
└─────────────────────────────────────────┘
           ↓
┌─────────────────────────────────────────┐

└─────────────────────────────────────────┘

      │

         └→ Сохранить только в localStorage
            └→ Перейти на index.html

## 🐛 Если что-то не работает

### Проблема: Камера просит разрешение постоянно

**Решение:**
- Убедитесь что даете разрешение при первом запросе
- Если "Заблокировано" - удалите сайт из запретов браузера
- Обновите страницу (F5)

### Проблема: Селфи черное/пустое

**Это может быть:**
- Камера еще загружается (добавьте задержку)
- Светлое окружение пикает в камеру
- Камера фронтальная, но вы смотрите в обратную сторону

**Решение:**
- Попробуйте другой браузер
- Проверьте что камера работает в другом приложении
- Посмотрите логи в консоли (F12)

### Проблема: Удаление все еще удаляет всю галерею

**Это ИСПРАВЛЕНО в текущей версии!**
- Убедитесь что вы загрузили последние изменения
- Откройте консоль (F12) и ищите логи удаления
- ID должны начинаться с 'selfie-'

## 📊 ID формат селфи

**Старый (неправильный):**
```
local-1710412200000-xyz123
(использовался только в localStorage)
```

**Новый (унифицированный):**
```
selfie-1710412200000-xyz123
(используется везде: сервер и localStorage)
```

Это исправляет проблему синхронизации между устройствами!
